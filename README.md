## 安装

确保您的电脑已经安装 Git。

```bash
git clone https://github.com/TATENcn/Home-Pages.git
```

## 使用

编辑 `config.json` 文件即可。

对于初学者，我们建议您阅读我们撰写的[指南](https://github.com/TATENcn/Home-Pages/blob/main/HowToDeployToGithubPages.md)以学会配置 Github Pages。
